Introduction

Sometimes it takes only 2 people to create an amazing idea. But sometimes you need a third one to visualize it - this when I come in as a UI designer.

The idea is to create this amazing website similarly to already existing one’s like Fiverr but mainly focused on freelancers.

My Role

When I joined this project some coding has been done and visual representation already existed so my job was to upgrade it and make designs for pages that haven’t been made yet.

I collaborated with 2 people. One being senior web developer (Mārtiņš Starķis) and product owner (Arnis Libenzons).

Problem

Web page has been partially made without assistance of UI designer therefore it looks visually unpleasing for an eye.

Color palette

Colors that are used in this website aren’t based on color theory which makes website look unprofessional or even make users uncomfortable to trust this site.

Layout

Edit icons shows up without following any patter or alignment.

Goals

  • To make website look more pleasing to eye and professional
  • To give developers visual representation of website so they don’t spend time on features that might actually look bad or be unimportant for MVP

Final thoughts

This project made me go trough a lot of ups and downs.

The biggest “up” about this project - once it’s ready and people are going to use it I’ll be happy that I participated and made some impact, possibly made sure that people like how it looks and how it works.

The biggest “down” - I had no guidance in designing process I was on my own. I would like to have some team member to help me out or to teach me something I didn’t know.
