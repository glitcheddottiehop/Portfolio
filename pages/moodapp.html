Introduction

My first project. I decided to create my first design based on apps that I have used a lot. Something that is familiar to me.

I couldn’t call MoodApp “the best app ever” but I believe it has it’s charm.

My Role

Even though it’s my first project I built while studying UI, I returned to this project after I finished course and did some upgrades.

Problem

After finishing the course and going through this project with more knowledge I realized that I have several problems:

Color palette

I doesn’t have a specific color palette. Colors in this app does not follow any color scheme.

Layout

All elements are placed randomly without following any pattern, alignment or spacing.

Components and naming

Elements are not made using components. In fact - doesn’t have any components and it contains groups with bad naming.

Goals

  • Create components
  • Create design system
  • Add correct and repetitive spacings
  • Give better names

Final Designs

Home

Changed black color to Neutral 900 to make it less intense.

Also changed button colors to triad color scheme to make it stand out more.

Mood added

Made changes to font color and icon thickness because previous version has bad contrast

Mood deleted

Made changes to font color and icon type because previous version has bad contrast and it can confuse user when using the same icon as for mood adding

Chart

Changed mood record’s colors to more contrasting ones

Mood records

Decided to hide action icons and make them visible on drag

Edit record

Nothing much changed here - following the same color palette and made text area wider and shorter

Impact

The moment I gave some users to try out new design prototypes I received these feedbacks:

  • “I like that colors on buttons are much brighter now they really do call for action”
  • “I would like to have it as an actual app”
  • “Icons makes more sense now”

Final thoughts

When I made this design first time I was confident that it looks great but the more I learned the more I started to not like my initial design. I started to spot a lot of errors. I even started to “cringe” but there’s saying “If you cringe it means you have grown”
