Introduction

Sometimes it takes only 2 people to create an amazing idea. But sometimes you need a third one to visualize it - this when I come in as a UI designer.

The idea is to create this amazing app which can help train your brain in a fun way. Game is about math where you have to get to specific sum.

My Role

When I joined this project some coding has been done but it had no planned out design to it.

I collaborated with 2 people. One being senior developer (Mārtiņš Starķis) and product owner (Arnis Libenzons).

Problem

App has been partially made without assistance of UI designer therefore it looks visually unpleasing for an eye.

Color palette

Colors that are used in this app aren’t based on color theory which makes app look unprofessional and not interesting.

Background

Background is plain color which also is not interesting.

Animations

App has no animations on user interaction. More animations could make app more “alive”.

Goals

  • Make app look more professional and interesting
  • Change app’s background
  • Add animations

Final Designs

New game

Added custom made background picture and stylized buttons in 2 variations.

I chose green as already selected buttons and yellow as options.

Level successful

Added animation on completing level successfully

Level failed

Added animation on failing level

Impact

The moment I gave some users to try out new design prototypes I received these feedbacks:

  • “New background looks better than solid color”
  • “It’s interesting”
  • “I like version with the roads”

Prototype

To try out prototype please visit this web site from PC

Final thoughts

I encountered such problems as creating components correctly. As I didn’t make them correctly I spent more time than needed to make changes in design.

I also had problems with background design since I wasn’t sure if moving background is a good option. I had idea about background that changes color all the time but it also could distract user from game itself.

Overall - I see a lot to work on here. Maybe more advanced animations and some changes in background.
